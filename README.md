# OpenCV 图像特征识别

说明：
- 环境：Windows + Git Bash，使用 Python 虚拟环境（venv）。
- 快速开始：
  1. 在项目根目录打开 Git Bash。
  2. python -m venv venv
  3. source venv/Scripts/activate
  4. pip install -r requirements.txt
  5. python src/test_env.py

目录结构（示例）：
- dataset/       # 存放图片（不要把大图提交到仓库）
- features/      # 保存提取的描述子（可 .npz）
- src/           # 源代码
- scripts/       # 辅助脚本
